# MNIST Handwritten Digit Classifier

A fully connected deep neural network trained on the MNIST dataset, 
deployed as an interactive web app.

## Results
- Training accuracy: 98%
- Test accuracy: 97.3%

## Techniques used
- TensorFlow / Keras
- Dropout regularization
- Early Stopping
- Deployed with TensorFlow.js in the browser

## How it works
Draw any digit (0–9) on the canvas and the model predicts it in real time.
